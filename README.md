# HRSword-Portable - [火绒剑](https://www.huorong.cn/)便携版

## Manual - 使用说明

1. Right-click on the `sysdiag.inf` file and select the `Install` option.

   右键选择 `sysdiag.inf` 文件后，选择 `安装` 选项。

2. Run `HRSword.exe` with administrator privileges.

   管理员权限启动 `HRSword.exe`
